# 回本计算器网页版展示版

目录：`D:\work\huiben-calculator-web`

## 文件结构

- `index.html`：页面结构
- `styles.css`：页面样式
- `script.js`：输入、计算、状态和速查逻辑

## 打开方式

### 方式一：直接打开

直接双击 `index.html`，即可在浏览器中预览。

### 方式二：本地静态服务

在当前目录启动任意静态服务后访问 `index.html` 即可，例如：

```bash
python -m http.server 8080
```

然后访问：`http://localhost:8080`

## 说明

网页版基于已验收的小程序版本实现，保持以下一致：

- 单页面、单功能
- 输入后实时计算
- 小数位最多两位
- `>=100` 时显示无法回本
- 速查点击带入
- 深色、克制、结果优先的视觉方向
