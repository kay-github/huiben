# 回本计算器 UI 设计提示词

- Product: 回本计算器
- Purpose: 提供给 UI 设计 AI 的设计提示词
- Source: 当前对话整理
- Last Updated: 2026-03-29

请为一个微信小程序设计一套**高保真 UI 界面**，产品名称是 **“回本计算器”**。

这是一个**极简、单功能、小而美**的工具类产品，唯一功能是：

用户输入“亏损幅度（%）”，系统立即计算并展示“回本涨幅（%）”。

计算公式为：

`回本涨幅 = (1 / (1 - 亏损幅度 / 100) - 1) * 100`

例如：
- 亏损 40%，回本涨幅 66.67%
- 亏损 50%，回本涨幅 100.00%
- 亏损 70%，回本涨幅 233.33%

请严格遵守以下设计要求，不要擅自增加功能：

## 产品定位

- 这是一个只做单一计算的小程序
- 卖点是：无广告、启动快、界面高级、随时能用
- 不要把它设计成金融资讯产品、投资社区、工具箱或多功能 App
- 不要增加登录、注册、会员、历史记录、分享裂变、广告位、推荐内容、资讯模块、行情图表

## 页面结构

请只设计一个主页面，首屏完成主要任务，结构如下：
1. 顶部标题区
2. 输入区：亏损幅度（%）
3. 结果区：回本涨幅（%）
4. 常用亏损档位速查区
5. 底部轻提示：仅作计算参考，不构成投资建议

## 交互要求

- 输入后实时计算
- 不需要“计算按钮”
- 可有清空输入按钮
- 常用亏损档位可点击带入输入框
- 输入为空时，结果区显示默认占位状态
- 输入大于等于 100% 时，显示“无法回本”的异常状态

## 视觉风格

请做出“国际互联网大厂精品工具”的感觉，方向参考：
- Apple 的克制高级
- Stripe 的精致渐变
- Linear 的暗色科技质感

设计气质要求：
- 极简
- 优雅
- 惊艳
- 专业可信
- 高级但不过度张扬

## 视觉表现要求

- 建议使用深色高级背景，如石墨黑、深海蓝黑、冷灰黑
- 可使用轻玻璃拟态或轻磨砂卡片质感
- 局部可有电光蓝、青蓝、微紫、暖金的高级强调色
- 保持足够留白
- 结果数字必须是视觉主角
- 页面看起来要像精品效率工具，不像普通表单页
- 不要做土味金融风，不要大面积红绿，不要廉价渐变，不要堆砌图标，不要花哨插画

## 内容文案

- 标题：回本计算器
- 副标题：亏损后，需要涨多少才能回本
- 输入标签：亏损幅度
- 输出标签：回本涨幅
- 底部提示：仅作计算参考，不构成投资建议
